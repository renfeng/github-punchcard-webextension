Are private repos supported?

Yes. Here's how.

  1. You must be a member of the private repository.
  2. Follow this link to acquire a new personal access token(*). Github will ask you to enter your password if you haven't do so for a while.
  3. The token can be reviewed and revoked, here.
    (*) Notes:
  • This web extension uses GitHub API v3, which requires authentication. Among the many implementation options, basic authentication via personal access tokens was chosen for its simplicity.
  • The token should be used solely by this web extension. And, it will be stored in Chrome sync, or
  • For Firefox a user must have Add-ons checked under the "Sync Settings" options in "about:preferences". See Firefox sync.
